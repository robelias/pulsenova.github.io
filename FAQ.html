404 Token Rules

$Nova404 is a brand new implementation for ERC404 fractional NFTs.
The token works with two mirrored contracts, each representing it's token standard. It was designed this way to allow for ease of use on both NFT marketplaces and DEXes. Upon minting, the contracts will emit 2 minting event for token $NOVA404.
The events are for the respective standards ERC20 and ERC721. This will be picked up by both any DEX or NFT marketplace, and will allow the token to be traded in both manners.

Upon NFT transfer, the held currency will move with it. Upon currency tranfer, the held NFT will move with it. As there is no way to peg an NFT individual ID to a currency fluid amount, the NFT will be burned and a new one will be minted in the receiving wallet.

The IDs burned and re-minted will persist. No new ids will be minted into the new wallet.
Meaning that there will only be 5555 NFTs in existence, numbered from 1 to 5555.

Utility?

$NOVA cores will be used as an inscription ticket for each season of the upcoming game Pulse Nova .
The token most be held in an inscription contract for the entire season, and the wallet that made the transaction will be marked as the identification of the player. This will be necessary to claim the game rewards when the season ends.
Additional utility will be added in the future. As the game evolves and seasons become longer and more complex.

Rarity?

Every $NOVA core will have unique DNA , the first season, the DNA will simply be used to calculate one of 11 shapes to display as picture for the NFT.
The same DNA will never change and can be reinterpreted each season to grant different perks (if any) and change the displayed metadata on marketplaces. Perks of each NFT will be visible one week before the season begins.
